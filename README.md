# Nested Comment

Simple example to build multi level comment in rails

# Setup

```
  rbenv install 2.3.1
  rbenv local 2.3.1

  gem install bundler -v 1.17.1
  bundler _1.17.1_ install

  bundle exec rake db:create
  bundle exec rake db:migrate

  rails s
```

# Rspec

```
  bundle exec rspec
```
